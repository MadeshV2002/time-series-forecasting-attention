# Time Series Forecasting with Attention

This project demonstrates time series forecasting using an LSTM model enhanced with an attention mechanism.

## Technologies Used
- Python
- NumPy
- Pandas
- TensorFlow / Keras
- LSTM
- Attention Mechanism

## Project Overview
- Data preprocessing and scaling
- Sequence generation for time series
- LSTM model with attention layer
- Model training and evaluation

## Use Case
This model can be applied to:
- Stock price prediction
- Sales forecasting
- Sensor data prediction

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the notebook or Python script
